# BambooBrush 2.0

A modern, bilingual (Swedish / English) storefront for BambooBrush — plastic-free, compostable
bamboo toothbrushes with boar bristles.

Static site — no build step. Just HTML, CSS and vanilla JavaScript.

## Features
- Responsive design with light/dark theme (time-based default: dark 20:00–06:00, light otherwise)
- SV / EN language switch
- Product gallery, cart drawer with free-shipping progress, FAQ, trust signals
- "Read more" microplastics modal
- Six policy pages (privacy, returns, terms, shipping, contact, legal)

## Run locally
Open `index.html` directly in a browser, or serve the folder with any static server.

## Hosting
Deployed as a static site via GitHub Pages.
